# 🖼️ Modern Wallpaper Gallery

A responsive wallpaper gallery built using **HTML**, **CSS**, and **JavaScript**. The application features real-time search, category filtering, and a modern glassmorphism user interface with an interactive lightbox.

---

## ✨ Features

- 🔍 Real-time image search
- 🏷️ Category-based filtering
- 🖼️ Interactive lightbox for image preview
- 📱 Fully responsive design
- 🎨 Modern Glassmorphism UI
- ⌨️ Keyboard navigation support
